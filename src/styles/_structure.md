# `src/styles/` Structure

Contains all vanilla CSS.
- `global.css`: The main stylesheet holding CSS variables for the color palette, typography definitions, and reset rules.
- **Aesthetic**: Warm whites (`#F9F8F6`), soft taupes (`#D6CFC7`), muted charcoals (`#2C2A29`), utilizing `Amarante` for Art Deco headlines and `Outfit` for modern body copy.
